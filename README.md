BMW sales analysis project with complete data cleaning, EDA, and interactive dashboards. Features Random Forest modeling for sales prediction, trend insights, customer behavior patterns, feature importance analysis, and well-designed visual reports to support data-driven decision-making and performance evaluation across models and years.

Here are some point focus on this project.

1. Project Overview: Analysis of BMW sales data to uncover key trends and insights.
2. Data Cleaning: Handling missing values, duplicates, outliers, and formatting issues.
3. EDA: Visual exploration of sales patterns, model performance, and yearly trends.
4. Feature Engineering: Creating meaningful variables for better predictions.
5. Dashboard: Interactive dashboards showing KPIs, trends, and customer insights.
6. Machine Learning Model: Random Forest model for predicting sales outcomes.
7. Model Evaluation: Accuracy, feature importance, and performance comparison.
8. Visualizations: Charts showing trends, correlations, and sales distribution.
9. Insights: Findings about customer behavior, model popularity, and growth.
10. Conclusion: Summary of results with recommendations for better business decisions.
